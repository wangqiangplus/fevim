# vimer(一份个人vim配置文件)

## 安装

- 安装vundle
  - git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

- 配置文件
  - git clone https://github.com/wungqiang/vimer.git
  - mv ./vimer/vimrc ~/.vimrc && rm -rf ./vimer

- 安装插件
  - 进入vim
  - 运行命令:PluginInstall
